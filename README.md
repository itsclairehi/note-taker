  ## Description
  This is a simple note-taker application that lets you store ideas and tasks in an editable list that is then stored to a server and recalled upon reloading. 
   
  ## Technologies
  * html, CSS, JavaScript
  * node, express.js, uuid.js
  * hosted on heroku
